# **House of Hearts** - a content mod for the charity event



## About the project
This mod was developed by the Balatro University community for the American Heart Association's annual House of Hearts event in 2026!

There are **15 new Jokers** and **1 new Deck** that have been added following the heart and physical wellness theme.

Mod also features **15 Achievements**:
- 9 Certification achievements, made with speedrunning in mind.
- 6 Mastery achievements, for exploring the content in the endless mode.

## How to install

**Manual installation**
1. Install [lovely injector](https://github.com/ethangreen-dev/lovely-injector#manual-installation)
2. Install [Steamodded/smods](https://github.com/Steamodded/smods/wiki#i-want-to-play-with-mods)
3. Download the [latest release](https://github.com/PLagger7/House-of-Hearts/releases) and put it in your Mods folder (same as Steamodded).

... you can also find the mod on [Balatro Mod Manager](https://balatro-mod-manager.dasguney.com/).
